# live-event 赛事直播使用教程

`live-event` 用于生成赛事直播 M3U 列表。

## Docker 运行

```bash
docker run -d --name live-event --restart always -p 28989:8989 iptvlive/live-event:latest
```

访问直播列表：

```text
http://你的服务器IP:28989/list.m3u
```

例如：

```text
http://127.0.0.1:28989/list.m3u
```

## Docker Compose

```yaml
services:
  live-event:
    image: iptvlive/live-event:latest
    container_name: live-event
    restart: always
    ports:
      - "28989:8989"
```

启动：

```bash
docker compose up -d
```

## CDN 或反向代理地址

如果需要把列表里的播放地址替换成你的域名，可以访问：

```text
http://你的服务器IP:28989/list.m3u?prefix=https://你的域名
```

例如：

```text
http://127.0.0.1:28989/list.m3u?prefix=https://example.com
```

## Watchtower 自动更新

如果已经有 Watchtower，不需要再创建第二个。重建现有 Watchtower，并把
`live-event` 加入监控列表即可：

```bash
docker rm -f watchtower
docker run -d --name watchtower --restart always -v /var/run/docker.sock:/var/run/docker.sock containrrr/watchtower:latest live-event --cleanup --interval 3600
```

如果还需要同时监控其他容器，把容器名一起写在 `live-event` 后面即可。

## 手动更新

```bash
docker pull iptvlive/live-event:latest
docker rm -f live-event
docker run -d --name live-event --restart always -p 28989:8989 iptvlive/live-event:latest
```

## 常用检查

检查容器状态：

```bash
docker ps | grep live-event
```

查看日志：

```bash
docker logs -f live-event
```
